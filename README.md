# VoltageMeasure

개발 툴: Keil uVision 5

개발 언어: C언어

목적: STM32F407에서 FreeRTOS, uC/GUI를 활용한 미니 오실레이터 만들기

사용 방법:
1. 엔코더를 돌려 MCU의 ADC input 핀의 전압을 변화시키고 그것을 실시간으로 측정하여 TFT-LCD에 표시
